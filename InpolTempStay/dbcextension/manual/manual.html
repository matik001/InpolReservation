How it works:

This browser extension allows users to quickly fill out and submit web forms. It works by automatically detecting captcha codes on the page, sending a request to the Death By Captcha server for resolution, then applying the resolved captcha solution to fill-out relevant fields in the HTML code of the page. This will allow users to quickly receive a captcha response to be submitted into the form within an average timeframe of ~30 seconds.. As this solution is only valid for this short time period, users must be vigilant and aware of how much time they have left so that they can make sure their entries are submitted successfully and in a timely manner.

Credentials

Your Death By Captcha username and password. You may get it on your user dashboard on deathbycaptcha.com.

Enable extension

Enables HTML code analysis. The extension is enabled by default, this means it will inspect and detect automatically for captchas and add a button to solve the captchas found. You can disable the extension or some captcha types, which means that the extension will not inspect to find those.

Error handling

Try to send a request again if a solution from Death By Catpcha didn't work or extension failed to send captcha challenge to deathbycaptcha.com. Please note, if a website with a captcha challenge refuse to accept a captcha resolution from extension, it will refresh a page with captcha and extension will detect it as a new captcha page. Even if this option is not enabled, an extension will send a captcha request to deathbycaptcha.com.

Delay - You may set up a delay for repeat of captcha submission, indicating amount of seconds of delay.

Enabled / Solve automatically

Check options accordingly for every captcha type.

Enabled - Check a code of the page for this type of captcha.

Solve automatically - If set, the extension will send the captcha challenge to deathbycaptcha.com API for resolution when found. If you leave it unchecked, an extension will add a "Solve with DeathByCatpcha" button to this webpage. If the captcha is invisible, the button will be placed under form submission button.

Proxy

Use proxy - enable this option if you want a captcha to be solved via your proxy server.

Proxy type - indicate proxy type. We only support HTTP and HTTPS. We can't detect proxy type automatically so you have to indicate it.

login:password@123.456.789.123:9876 - indicate your proxy address, port and credentials accordingly. If your proxy doesn't need authorization, leave only IP and port: "123.456.789.123:9876"

Image captcha solving

The extension could not detect where is a captcha located and where exactly a resolution should be placed in when captcha is solved. So you have to indicate this for every website you will need to bypass captcha on. To do so Right click on a picture and choose "Solve this captcha" to let extension know where the captcha is located. Then do the same for resolution field, Right click on it and choose "Insert captcha resolution here". These parameters will be stored in the extension settings for this particular website.
